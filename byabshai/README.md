# ByabshAI - SME Cashflow Copilot (Phase-2 Prototype)

**What judges can do in 60-90s**
1. Upload two CSVs from `/samples` (sales + bank).
2. See KPIs instantly, click **Detect anomalies**, then **Compute readiness**.
3. Go to **Report**, **Download PDF**, and **Verify Hash**.

## Run locally

```bash
npm install
npm run dev
# open http://localhost:3000
```
